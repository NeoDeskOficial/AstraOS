# 🚀 AstraOS

> Sistema CRM + ERP + Help Desk desenvolvido pela NeoDesk Informática.

O AstraOS é uma plataforma web desenvolvida em PHP, MySQL, HTML, CSS e JavaScript, projetada para centralizar a gestão da NeoDesk Informática em um único sistema.

---

# 📌 Objetivos

- CRM (Gestão de Clientes)
- ERP (Gestão Empresarial)
- Help Desk
- Ordens de Serviço
- Financeiro
- Estoque
- Contratos
- RH
- Portal do Cliente
- Relatórios
- API REST (futuramente)
